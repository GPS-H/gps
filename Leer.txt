o optimizar el código. Lo que hice fue lo siguiente:

+index.php --> sera el login de usuarios.
	-Hace uso de clase_conexion.php --> que sera la clase que hare la conección con la BD
	-Hace uso de clase_login.php --> este dependera de clase_conexion.php, pues hara una consulta para dar el acceso;

+administrador.php
	-Hace uso de head.php --> y este tendrá el head base (css, js) 
	-Hace uso de header.php -->y este tendrá el menú base (cambiara el menu dependiendo del tipo de usuario que accedió)
	-Hace uso de cerrarSesión -->para salir del sistema
